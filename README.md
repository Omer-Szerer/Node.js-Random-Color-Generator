# Node.js Random Color Generator

An application that return a random color in the command line.

```
node index.js
          ###############################
          ###############################
          ###############################
          #####                     #####
          #####       #edff28       #####
          #####                     #####
          ###############################
          ###############################
          ###############################
```

```
node index.js
          ###############################
          ###############################
          ###############################
          #####                     #####
          #####       #56004f       #####
          #####                     #####
          ###############################
          ###############################
          ###############################
```

It should also be able to accept the request for hue (eg. green) and luminosity (eg. dark):

```
node index.js red
          ###############################
          ###############################
          ###############################
          #####                     #####
          #####       #b72d42       #####
          #####                     #####
          ###############################
          ###############################
          ###############################
```

```
node index.js blue dark

          #####                     #####
```
